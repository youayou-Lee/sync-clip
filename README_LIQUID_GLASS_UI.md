# 🌟 SyncClip 液态玻璃 UI 使用指南

## 概述

已成功将 SyncClip 的传统 Tkinter UI 替换为现代化的 PyQt6 液态玻璃风格界面，具有丰富的动画效果和美观的视觉设计。

## ✨ 主要特性

### 🎨 液态玻璃美学
- **半透明效果**: 所有组件都具有半透明背景，营造玻璃质感
- **模糊背景**: 模糊半径创建深度感
- **渐变色彩**: 动态渐变背景，色彩丰富
- **圆角设计**: 现代化的圆角边框设计

### 🌈 动画效果
- **涟漪效果**: 按钮点击时的水波纹动画
- **滑入动画**: 历史记录从左侧滑入，设备列表从右侧滑入
- **淡入效果**: 组件加载时的平滑淡入
- **悬停效果**: 鼠标悬停时的缩放和颜色变化
- **脉冲效果**: 在线设备的呼吸灯效果
- **交错动画**: 列表项依次出现的动画效果

### 🎯 交互体验
- **流畅过渡**: 所有状态变化都有平滑动画
- **视觉反馈**: 丰富的用户交互反馈
- **状态指示**: 实时的设备在线状态显示
- **响应式设计**: 自适应窗口大小变化

## 🚀 运行方法

### 基本运行
```bash
# 运行主应用程序
uv run python main.py
```

### 测试模式（无GUI环境）
```bash
# 使用offscreen平台插件测试
QT_QPA_PLATFORM=offscreen uv run python main.py

# 或者运行基本测试
QT_QPA_PLATFORM=offscreen uv run python test_ui_basic.py
```

## 🏗️ 技术架构

### 核心组件
- **ModernClipboardApp**: 主应用程序窗口
- **LiquidGlassWidget**: 液态玻璃效果基类
- **RippleButton**: 带涟漪效果的按钮
- **ClipboardHistoryItem**: 剪贴板历史记录项
- **DeviceItem**: 设备显示项

### 动画系统
- **SlideInAnimation**: 滑入动画类
- **FadeInAnimation**: 淡入动画类
- **PulseEffect**: 脉冲效果类
- **FloatingWidget**: 悬浮动画类

### 颜色方案
- **主背景**: 蓝紫色渐变 (#667eea → #764ba2)
- **玻璃效果**: 白色半透明 (rgba(255,255,255,180))
- **Windows设备**: 蓝色 (#3498db)
- **Linux设备**: 绿色 (#27ae60)
- **macOS设备**: 紫色 (#9b59b6)

## 📱 界面布局

### 顶部标题栏
- 应用图标和名称
- 刷新设备按钮
- 清空历史按钮

### 左侧：剪贴板历史
- 显示最近20条剪贴板记录
- 每项包含设备信息、时间戳、内容预览
- 支持文本和图片类型
- 一键复制功能

### 右侧：设备列表
- 实时显示连接的设备
- 设备状态指示（在线/离线）
- 平台标识和IP地址
- 最后在线时间显示

### 底部状态栏
- 实时运行状态
- 历史记录数量统计
- 设备事件提示

## 🎮 交互说明

### 基本操作
- **复制内容**: 点击历史记录项的"复制"按钮
- **清空历史**: 点击顶部"清空历史"按钮并确认
- **刷新设备**: 点击"刷新设备"按钮重新扫描

### 动画效果
- **悬停效果**: 鼠标悬停在历史记录或设备项上会看到缩放效果
- **点击反馈**: 按钮点击时会有涟漪动画扩散
- **加载动画**: 新的历史记录和设备会以动画形式出现

### 快捷键
- **Ctrl+L**: 清空剪贴板历史（保留原有功能）

## 🔧 自定义配置

### 动画参数
可以在代码中调整以下参数：
- 动画持续时间 (`duration`)
- 模糊半径 (`blur_radius`)
- 透明度 (`opacity`)
- 动画缓动曲线 (`easing_curve`)

### 颜色主题
可以修改以下颜色值：
- 背景渐变色
- 设备状态颜色
- 悬停效果颜色
- 脉冲效果颜色

## 🐛 故障排除

### 常见问题
1. **无法启动GUI**: 确保系统已安装必要的GUI库
   ```bash
   sudo apt-get install libxcb-cursor0
   ```

2. **动画卡顿**: 检查系统性能，可以降低动画频率

3. **显示异常**: 尝试使用不同的平台插件
   ```bash
   QT_QPA_PLATFORM=wayland uv run python main.py
   ```

### 调试模式
启用详细日志输出：
```bash
uv run python main.py --debug
```
